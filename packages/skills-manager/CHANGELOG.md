# Changelog

## 0.1.0

Initial release. Programmatic skill workspace + agent-link manager built
around seven primitives — `add`, `link`, `unlink`, `remove`, `listSkills`,
`listLinks`, `rescan` — with a manifest+filesystem hybrid as the source of
truth.

Supports 53 agents catalogued by [`vercel-labs/skills`](https://github.com/vercel-labs/skills).
